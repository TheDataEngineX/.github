# Support

Need help with DataEngineX? Here's where to find it.

## Documentation

- [DEX Docs](https://docs.thedataenginex.org) — Core framework
- [DEX Studio Docs](https://github.com/TheDataEngineX/dex-studio) — Web UI
- [InfraDEX Docs](https://github.com/TheDataEngineX/infradex) — Infrastructure

## Getting Help

### GitHub Discussions

For questions, ideas, and community support:
[github.com/orgs/TheDataEngineX/discussions](https://github.com/orgs/TheDataEngineX/discussions)

### Issue Tracker

For bugs and feature requests, use the issue tracker in the relevant repo:

| Repo | Issues |
|------|--------|
| DEX | [Issues](https://github.com/TheDataEngineX/DEX/issues) |
| dex-studio | [Issues](https://github.com/TheDataEngineX/dex-studio/issues) |
| infradex | [Issues](https://github.com/TheDataEngineX/infradex/issues) |

## What NOT to Use Issues For

- General questions → use Discussions
- Security vulnerabilities → see [SECURITY.md](SECURITY.md)
- Feature requests without context → use the Feature Request template

## Response Times

This is an open-source project maintained by volunteers. We aim to respond to
issues within a week, but response times may vary.
